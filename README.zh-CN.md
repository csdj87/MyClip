
# <img src="myclip.svg" width="24"> MyClip

一款win剪贴板历史工具软件，支持固定、类型显示、全局搜索、主题切换等。

## 🌟 功能特点

- 🎨 支持图片、富文本、文件路径剪贴板历史。
- 🔍 剪贴板历史全文搜索
- 🌗 支持夜间主题、主题自动、手动切换
- 🎹 自定义全局热键
- 📌 固定剪贴板历史
- 🧾 sqlite本地数据库
- 💡 完全开源
